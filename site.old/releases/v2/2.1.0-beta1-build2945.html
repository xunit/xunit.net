Release Notes: March 21, 2015

xUnit.net v2 2.1 Beta 1

Core framework

  • Added support for ASP.NET 5 Beta 3.

Extensibility

  • TestInvoker.InvokeTestMethodAsync was accidentally made public just before the 2.0 release. It has been restored back to protected, as was originally intended.
  • TestCollection.Serialize and TestCollection.Deserialize has been made virtual, so developers can re-use the implementation while enhancing the serialization behavior.
  • BUG: TestInvoker would invoke unit tests even if the user had requested cancellation. This has been fixed.

Visual Studio Runner 2.1 Beta 1

For information on using xUnit.net from within Visual Studio, see Running xUnit.net tests in Visual Studio.

  • There were no issues fixed for this release.

ASP.NET Runner 2.1 Beta 1

  • Added support for ASP.NET 5 Beta 3.
  • The ability to run tests from inside of Visual Studio 2015 is back. This has been validated with VS 2015 CTP 6.
  • A build.sh script (supporting Linux and OS X) has been added.
