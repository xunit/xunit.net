Release Notes

These release notes are a comprehensive list of changes from 2.6.4 to 2.6.5.

Core Framework

  • We have converted the TheoryData base class to IReadOnlyCollection<object[]> instead of just IEnumerable<object[]>, which means you can now index the items and get an item count. Additionally, we've added two constructors to TheoryData<T> (one taking IEnumerable<T> and one taking params T[]) and an AddRange method so that it can more easily be used with array-style initializers. xunit/xunit#2854
