Release Notes

These release notes are a comprehensive list of changes from 2.6.3 to 2.6.4.

Assertion library

  • BUG: Fixed an issue introduced with the preference for IEquatable<T> over IEnumerable<T> when the value types in a dictionary were of differing types. This manifested as an InvalidCastException being thrown while trying to compare values in the dictionary against each other. xunit/xunit#2850

Runner utility

  • BUG: Fixed a race condition introduced by ExecutionSink in 2.6.3 which was causing runners to sometimes trigger an ObjectDisposedException while finishing running an assembly. This was more easily reproduced in low CPU thread count conditions (more common in CI environments than developer machines). xunit/visualstudio.xunit#396
