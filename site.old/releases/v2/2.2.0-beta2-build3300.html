Release Notes: June 27, 2016

xUnit.net v2 2.2 Beta 2

Core framework

  • Adds support for .NET Core 1.0 RTM.
  • Adds support for .NET Core SDK 1.0 Preview 2.
  • Adds support for the new netstandard target frameworks. To use these new NuGet packages, you will need NuGet 3.4+ (for VS2015) or NuGet 2.12+ (for VS2012 and VS2013).
  • BUG: Improved reliability when reporting to AppVeyor with a longer timeout.

Extensibility

  • Moved IExecutionVisitor and XmlAggregateVisitor into xunit.runner.utility; if you were previously hand-copying these files into your runner, you can remove your copy now in favor of the one that ships in the runner DLL.

.NET CLI Runner 2.2 Preview 2

For information on using xUnit.net with the .NET CLI, see Getting started with xUnit.net (.NET Core / ASP.NET Core)

  • Adds support for .NET Core 1.0 RTM.
  • Adds support for .NET Core SDK 1.0 Preview 2.

Visual Studio Runner 2.2 Beta 2

For information on using xUnit.net from within Visual Studio, see Running tests with Visual Studio in the Getting Started documentation.

  • Support for running PCL-based unit test libraries within Visual Studio has been removed. It is recommended that all unit tests target a specific device, and any shared code can be reused through the shared library feature in Visual Studio.
