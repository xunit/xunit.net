Release Notes for Analyzers: May 6, 2023

xUnit.net Analyzers 1.2.0 Pre-Release (build 7)

Usage Analyzers

  • Fixed an issue with xUnit1025 where the analyzer was throwing an exception with specific input data (see the attached bug). xunit/xunit#2662

Code Fixers
